# Jeeto Jee Mentorship Platform

A Flask-based web application for the Jeeto Jee Mentorship Program.

## Features
-   **Landing Page**: Hero section, mentors, features, testimonials, and FAQ.
-   **Payments**: Integrated Razorpay for course purchases.
-   **Authentication**: Admin dashboard and user login.
-   **Contact**: Integrated Gmail and WhatsApp Channel links.

## Tech Stack
-   **Backend**: Python, Flask, SQLite
-   **Frontend**: HTML, CSS, JavaScript
